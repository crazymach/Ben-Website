WhichWay (Innovation Quest Finalist)

WhichWay is the first application that uses pace and compass calibration to track user position without the use of GPS or Data Service - or any external connection at all for that matter. The application uses the accelerometer to detect user movement and the compass for direction. The user inputs initial calibration settings, and then the application tracks their movement from thereon. QuadCopter


This project was a summer project of 2015. Starting with a QAV 250, I learned about the necessary components of the Quadcopter. This is the first project working with ESC (Electronic Speed Controllers) and radio transmitters/receivers. Working on the Quadcopter including alot of soldering bullet connectors to the ESC and Brushless Motors. The Quadcopter utilized a Power Distribution board to distribute power to all 4 motors equally. This was all controlled through a transmitter wired to a CC3D flight controller which was in charge of balancing the motors so that the Quadcopter can be level. Overall, I was introduced to radio controllers since I had to bind the transmitter and receiver, and how to break down a project into seperate subcomponents.

Sumo Bot


Hosted by the Robotics Club of Cal Poly San Luis Obispo, this is my first experience working on a Arduino with a team. As a introduction to robotics, I learned the fundementals of soldering and how to wire and code an Arudino. Starting with very basic examples, I was introduced to the Arduino IDLE and how to do simple blink examples. From there, the lessons moved onto how to read sensor values through the IDLE Serial Monitor. By wiring a light sensor and tweaking the values, my team and I were able to interprept the data and have the robot sense the black boundary line. From there, the team wired the motors and battery, updated the code to include a reverse function when the bot hit a black line, and we finally had a working Sumo Bot.

Fabric Pressure Sensor


The Fabric Sensor was one of the first projects I've done with Arudino. Using a Piezoresistive known as Velostat, I was able to create this sensor. By sewing the Velostat with conductive thread, I was then able to take in data through the arduino and some alligator clips using the Arduino IDLE Serial Monitor.
