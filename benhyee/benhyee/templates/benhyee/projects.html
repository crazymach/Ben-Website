WhichWay (Innovation Quest Finalist)

WhichWay is the first application that uses pace and compass calibration to track user position without the use of GPS or Data Service - or any external connection at all for that matter. The application uses the accelerometer to detect user movement and the compass for direction. The user inputs initial calibration settings, and then the application tracks their movement from thereon.
